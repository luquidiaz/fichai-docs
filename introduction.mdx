## Bienvenido a FichAI API

La API de FichAI te permite acceder a información de desarrollos inmobiliarios listados en nuestro MLS (Multiple Listing Service). Podés consultar desarrollos, sus unidades disponibles, precios y más.

## Base URL

Todas las requests a la API deben hacerse a:

```
https://app.fichai.com/api/public
```

## Formato de respuestas

La API devuelve respuestas en formato JSON. Todas las respuestas exitosas tienen código `200 OK`.

```json
{
  "desarrollos": [...],
  "meta": {
    "total": 100,
    "limit": 50,
    "offset": 0,
    "has_more": true
  }
}
```

## Rate Limiting

La API tiene límites de requests para garantizar un servicio estable:

- **100 requests por minuto** por API key (por defecto)
- Los headers `X-RateLimit-*` indican tu uso actual

<Note>
  Si necesitás un rate limit mayor, contactanos a soporte@fichai.com
</Note>
