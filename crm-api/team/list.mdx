## GET /api/v1/team

Endpoint de descubrimiento que retorna los miembros de tu organizacion. Usa este endpoint para obtener los valores de `assigned_to` (UUID) o `assigned_to_email` necesarios para asignar leads.

## Autenticacion

```bash
Authorization: Bearer sk_live_YOUR_API_KEY
```

**Scope requerido:** `contacts:read`

## Comportamiento segun rol

<Warning>
  El comportamiento varia segun el tipo de API key:
</Warning>

| Tipo de key | Resultado |
|-------------|-----------|
| **Owner/Admin** | Retorna **todos** los miembros de la organizacion |
| **Member** | Retorna **solo al creador** de la key (a si mismo) |

## Respuesta

<ResponseField name="data" type="array">
  Array de miembros del equipo.

  <Expandable title="Propiedades de cada miembro">
    <ResponseField name="id" type="string">UUID del usuario (usar como `assigned_to` en `POST /leads`)</ResponseField>
    <ResponseField name="name" type="string">Nombre completo</ResponseField>
    <ResponseField name="email" type="string">Email (usar como `assigned_to_email` en `POST /leads`)</ResponseField>
    <ResponseField name="role" type="string">Rol: `owner`, `admin`, o `member`</ResponseField>
  </Expandable>
</ResponseField>

<Tip>
  Para configuraciones simples en Zapier/n8n, usa `assigned_to_email` en vez de `assigned_to` (UUID). Es mas facil de configurar y recordar.
</Tip>

<RequestExample>
```bash cURL
curl -X GET "https://app.fichai.com/api/v1/team" \
  -H "Authorization: Bearer sk_live_YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://app.fichai.com/api/v1/team', {
  headers: {
    'Authorization': 'Bearer sk_live_YOUR_API_KEY'
  }
});

const { data: team } = await response.json();

// Encontrar un agente por email
const agent = team.find(m => m.email === 'agente@miempresa.com');
console.log('Agent ID:', agent.id);
```

```python Python
import requests

response = requests.get(
    'https://app.fichai.com/api/v1/team',
    headers={'Authorization': 'Bearer sk_live_YOUR_API_KEY'}
)

team = response.json()['data']
for member in team:
    print(f"{member['name']} ({member['role']}) - {member['email']}")
```
