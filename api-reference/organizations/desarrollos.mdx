---
title: 'Desarrollos por Organización'
api: 'GET /organizations/{slug}/desarrollos'
description: 'Obtiene todos los desarrollos públicos de una organización'
---

## Descripción

Retorna información de una organización (inmobiliaria) junto con todos sus desarrollos públicos listados en el MLS de FichAI.

## Parámetros

<ParamField path="slug" type="string" required>
  Slug único de la organización. Es el identificador URL-friendly de la inmobiliaria.

  Ejemplo: `inmobiliaria-premium`
</ParamField>

<ParamField query="country" type="string">
  Código de país ISO para filtrar desarrollos.

  Ejemplos: `AR`, `UY`, `MX`
</ParamField>

<ParamField query="status" type="string" default="disponible,en_construccion">
  Estados a incluir, separados por coma.

  Valores posibles: `disponible`, `en_construccion`, `vendido`, `pausado`
</ParamField>

## Respuesta

<ResponseField name="organization" type="object">
  Información de la organización.

  <Expandable title="Propiedades">
    <ResponseField name="id" type="string">
      UUID de la organización
    </ResponseField>
    <ResponseField name="name" type="string">
      Nombre de la organización
    </ResponseField>
    <ResponseField name="slug" type="string">
      Identificador URL-friendly
    </ResponseField>
    <ResponseField name="logo_url" type="string">
      URL del logo
    </ResponseField>
    <ResponseField name="website" type="string">
      Sitio web de la organización
    </ResponseField>
    <ResponseField name="description" type="string">
      Descripción de la organización
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="desarrollos" type="array">
  Array de desarrollos de la organización. Misma estructura que en `/desarrollos`.
</ResponseField>

<ResponseField name="meta" type="object">
  Metadata

  <Expandable title="Propiedades">
    <ResponseField name="total" type="integer">
      Total de desarrollos de la organización
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://app.fichai.com/api/public/organizations/inmobiliaria-premium/desarrollos" \
  -H "Authorization: Bearer pk_live_xxxxxxxxxxxx"
```

```javascript JavaScript
const response = await fetch(
  'https://app.fichai.com/api/public/organizations/inmobiliaria-premium/desarrollos',
  {
    headers: {
      'Authorization': 'Bearer pk_live_xxxxxxxxxxxx'
    }
  }
);

const { organization, desarrollos } = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://app.fichai.com/api/public/organizations/inmobiliaria-premium/desarrollos',
    headers={'Authorization': 'Bearer pk_live_xxxxxxxxxxxx'}
)

data = response.json()
organization = data['organization']
desarrollos = data['desarrollos']
```
</RequestExample>
