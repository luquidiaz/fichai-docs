## Base URL

```
https://app.fichai.com/api/public
```

## Autenticación

Todas las requests requieren una API key en el header `Authorization`:

```bash
Authorization: Bearer pk_live_xxxxxxxxxxxx
```

## Formato de respuestas

Todas las respuestas son JSON. Las respuestas exitosas tienen código `200 OK`.

## Convenciones

- **IDs**: UUIDs v4
- **Fechas**: ISO 8601 (`2025-12-01T00:00:00Z`)
- **Monedas**: Códigos ISO 4217 (`USD`, `ARS`, `UYU`)
- **Países**: Códigos ISO 3166-1 alpha-2 (`AR`, `UY`, `MX`)
