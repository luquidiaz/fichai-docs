## Obtené tu API Key

<Steps>
  <Step title="Contactá a FichAI">
    Para obtener una API key, contactá a nuestro equipo en **soporte@fichai.com** indicando:
    - Nombre de tu empresa/proyecto
    - Uso que le darás a la API
    - Volumen estimado de requests
  </Step>
  <Step title="Recibí tu API Key">
    Te enviaremos una API key con el formato:
    ```
    pk_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
    ```
    <Warning>
      Guardá tu API key en un lugar seguro. No la compartas públicamente ni la incluyas en código del lado del cliente.
    </Warning>
  </Step>
  <Step title="Hacé tu primera request">
    Probá tu API key con una request simple:
    ```bash
    curl -X GET "https://app.fichai.com/api/public/desarrollos?limit=5" \
      -H "Authorization: Bearer pk_live_tu_api_key"
    ```
  </Step>
</Steps>

## Tu primera request

### Con cURL

```bash
curl -X GET "https://app.fichai.com/api/public/desarrollos" \
  -H "Authorization: Bearer pk_live_xxxxxxxxxxxx" \
  -H "Content-Type: application/json"
```

### Con JavaScript/Node.js

```javascript
const response = await fetch('https://app.fichai.com/api/public/desarrollos', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer pk_live_xxxxxxxxxxxx',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data.desarrollos);
```

### Con Python

```python
import requests

headers = {
    'Authorization': 'Bearer pk_live_xxxxxxxxxxxx',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://app.fichai.com/api/public/desarrollos',
    headers=headers
)

data = response.json()
print(data['desarrollos'])
```

### Con PHP

```php
<?php
$ch = curl_init();

curl_setopt_array($ch, [
    CURLOPT_URL => 'https://app.fichai.com/api/public/desarrollos',
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer pk_live_xxxxxxxxxxxx',
        'Content-Type: application/json'
    ]
]);

$response = curl_exec($ch);
$data = json_decode($response, true);

print_r($data['desarrollos']);
?>
```
